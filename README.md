# Eksplorasi Regresi Linear Sederhana (Simple Linear Regression)

Repository ini berisi hasil eksplorasi praktikum Machine Learning mengenai **Supervised Learning**, khususnya penerapan algoritma **Simple Linear Regression** menggunakan bahasa pemrograman Python.

## 📌 Deskripsi Tugas
Eksplorasi ini dibagi menjadi dua bagian utama untuk memahami konsep dasar regresi linear:
1. **Modul 1: Data Sembarang** Membuat model regresi linear sederhana menggunakan data *dummy* berupa data penjualan dan harga untuk melihat sejauh mana harga dipengaruhi oleh angka penjualan.
2. **Modul 2: Dataset Asli (FuelConsumptionCo2)** Mengimplementasikan model regresi linear pada dataset nyata dari IBM Object Storage. Pada tahap ini, model dilatih untuk memprediksi jumlah emisi CO2 (`CO2EMISSIONS`) berdasarkan ukuran mesin kendaraan (`ENGINESIZE`).

## 🛠️ Teknologi yang Digunakan
Eksplorasi ini dibangun menggunakan beberapa *library* utama Python:
* **NumPy**: Untuk komputasi array dan perhitungan saintifik.
* **Pandas**: Untuk manipulasi dan analisis data (membaca file CSV).
* **Matplotlib**: Untuk visualisasi data ke dalam bentuk grafik *scatter plot* dan garis regresi.
* **Scikit-Learn (sklearn)**: Untuk memanggil dan melatih model `LinearRegression`.
